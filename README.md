# 雪華（sekka）
漢字を主要パーツで分類し、送り仮名への連接を考慮した漢字直接入力

- カナ配列部分は「薙刀式」を想定
- 実装はLinuxのみ

## 目的
IMEがなくてもLinuxで日本語を打てるようにする（防災用のキャンプ道具的な位置付け）

## 特徴
- 4000字を収録
- 同時3打、または4打で漢字1字を入力
- 異体字同置
- 頻出熟語は省打鍵入力も可

## 必要なもの
- KMonad 4.2以上
